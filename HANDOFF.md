The H-O 2.9.4(ii) proof requires:
1. For a ∈ P₁(e), the algebra ℝ[a] = Span{e, a, a², ...} is:
   - Commutative (Jordan)
   - Associative (jpow_add gives a^m ∘ a^n = a^{m+n})
   - No nilpotents (formal reality + no_nilpotent_of_formallyReal)
   - Finite-dimensional → Artinian

3. Each field has an identity which is an idempotent in P₁(e)
4. By `primitive_idempotent_in_P1`, only trivial idempotents → single field
5. Formally real field = ℝ (eliminate ℂ)
6. Hence dim P₁(e) = 1

**Key challenge:** Constructing CommRing instance on the subalgebra generated by one element.

## Key Learnings

1. **jpow a 0 = jone ∉ P₁(e)** in general (only if e = jone). So powers in P₁(e)
   only work for n ≥ 1, hence the helper is `jpow_succ_mem_peirce_one`.

2. **Subalgebra approach:** The H-O proof works with the algebra generated by a
   SINGLE element, not all of P₁(e). This single-element algebra IS associative
   (power associativity), whereas P₁(e) as a whole is NOT associative for
   non-primitive idempotents.
